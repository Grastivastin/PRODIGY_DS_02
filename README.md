# PRODIGY_DS_02

## Task 02: Exploratory Data Analysis (EDA)

### Objective
To perform data cleaning and exploratory data analysis on the Titanic dataset to identify patterns and relationships between variables.

### Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

### Dataset
- Titanic Dataset

### Data Cleaning
- Filled missing Age values using median
- Filled missing Embarked values using mode
- Dropped Cabin column due to excessive missing values

### Key Insights
- Females had a higher survival rate than males
- Passengers in higher classes survived more
- Most passengers were between 20â€“40 years old
- Survival was influenced by gender and passenger class
